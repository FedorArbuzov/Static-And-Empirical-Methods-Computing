Функция распределения:

\[
 F(x) = 
  \begin{cases} 
   \ 0 & \text{if } x < 0 \\
   \ \frac{(1 - e^{-x})}{(1 - e^{-1})} & \text{if } 0 < x < 1 \\
   \ 1 & \text{if } x < 0
  \end{cases}
\]
Тогда функция квантилей:
\[
